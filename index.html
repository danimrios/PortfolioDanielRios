SERVICIO SQL CINE

El proyecto consiste en la creación de una base de datos relacional para un cine. Esta base de datos está diseñada para gestionar eficientemente la programación de películas, la administración de localidades y salas, así como el seguimiento de ingresos por ventas de entradas y productos de consumo. se busca cumplir de la manera mas optima posible los principios de interidad de los datos, independencia, atomicidad, consistencia, aislamiento y durabilidad. el sistema gestor cuenta con
Stored Procedures: como ejemplo SP_Eliminar_Pelicula: Este procedimiento almacenado realiza una baja lógica de una película en la tabla Películas y actualiza el estado en la tabla Cartelera_Pelicula
Triggers: como ejemplo TR_UpdateStock:Este trigger se activa después de insertar una nueva venta en la tabla DETxFOOD y actualiza el stock de los productos en Catalogo_Food. Si el stock es insuficiente, revierte la transacción.
